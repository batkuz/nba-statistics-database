# NBA Statistics Database Project
This project is a relational database built using Microsoft SQL Server.

## Features
- Teams table
- Players table
- Games table
- Player game statistics
- Analytical queries (total points, averages, team performance)

## Technologies
- SQL Server
- SSMS
- Relational Database Design
- JOIN operations
- GROUP BY aggregation
- Foreign Keys & Constraints

## Purpose
Demonstrates database design, normalization, data relationships, and analytical querying skills.
